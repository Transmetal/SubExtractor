# SubExtractor

Subdomain Enumeration Tool

<h3> Mode Of Execution: </h3>

* apt-get install python3

* apt-get install git

* git clone https://github.com/AngelSecurityTeam/SubExtractor

* cd SubExtractor

* pip3 install requests

* python3 subextractor.py
